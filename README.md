# Klasifikasi Sampah Otomatis

Aplikasi untuk mengklasifikasikan jenis sampah menggunakan Deep Learning.

## Fitur
- Upload gambar sampah
- Klasifikasi otomatis
- Confidence score
- Top 5 prediksi

## Akurasi Model
- Transfer Learning (MobileNetV2): 94.42%

## Cara Menjalankan
1. Install dependencies: `pip install -r requirements.txt`
2. Jalankan: `streamlit run app.py`
